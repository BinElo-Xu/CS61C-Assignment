1. Use "set args" followed the args to set the arguments, when you type "run", then you will use the arguments you have just set.
And if you subsequently "run" without any arguments, the same set of arguments will be reused. And also, you can use "run arg1 arg2"
to set the arguments, but this time args will not be stored, you must set the arguments next time when you "run".
2. Use "b functionName" to set a function break point, "b lineNumber" to set a line break point and "b address" to set the address
break point.
3. Also you can use "s" to execute c code step by step, and this time you will go into the function.
4. Use "n" to execute next line of code and this commond will not go into the function.
5. Type "run" commond when you stopping at a break point.
6. Use "p expr" to display the value of expression expr.
7. Use "display expr" to display the value of expression expr after each step automatically.
8. Use "info locals" to display all local variables(current function) name and their value.
9. Use "q" commond to quit the gdb session.
